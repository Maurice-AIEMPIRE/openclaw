# Research Sources

## Primary (check every heartbeat)

- **GitHub Trending** — `https://github.com/trending?since=daily` — New repos in AI/ML
- **Hacker News** — `https://news.ycombinator.com` — Top stories tagged AI
- **Hugging Face** — `https://huggingface.co/models?sort=trending` — New model releases

## Secondary (check weekly)

- **Reddit r/LocalLLaMA** — `https://reddit.com/r/LocalLLaMA` — Local model community
- **Reddit r/MachineLearning** — `https://reddit.com/r/MachineLearning` — Research papers
- **ArXiv cs.AI** — `https://arxiv.org/list/cs.AI/recent` — Latest papers
- **Twitter/X AI community** — Key accounts posting about open source models

## Agent / Automation specific

- **OpenClaw releases** — `https://github.com/openclaw/openclaw/releases`
- **LM Studio releases** — `https://lmstudio.ai`
- **Ollama releases** — `https://github.com/ollama/ollama/releases`

## Reliability rating

- **Tier 1 (trust)**: GitHub repos with code + benchmarks, ArXiv papers, official model cards
- **Tier 2 (verify)**: Blog posts with code, Reddit posts with reproduction steps
- **Tier 3 (skeptical)**: Twitter threads, marketing pages, "100x" claims without evidence
